# 2배 레버리지 거래 지표 조정 가이드 📊

## 개요

2배 레버리지 거래는 수익과 손실이 모두 2배로 증폭되므로, 기술적 지표의 **민감도를 높이고** **리스크 임계값을 낮춰야** 합니다. 이 문서는 레버리지 거래를 위한 지표 조정 방법을 상세히 설명합니다.

---

## 주요 기술 지표 상세 설명

### 1. RSI (Relative Strength Index) - 상대강도지수

#### 📌 지표 의미
- **범위**: 0 ~ 100
- **목적**: 가격 변동의 강도를 측정하여 과매수/과매도 상태 판단
- **계산**: 일정 기간 동안의 상승폭과 하락폭의 비율

#### 💡 해석 방법
- **70 이상**: 과매수 구간 → 매도 압력 증가 가능성
- **30 이하**: 과매도 구간 → 매수 압력 증가 가능성
- **50 근처**: 중립 구간
- **다이버전스**: 가격과 RSI의 방향이 다를 때 추세 전환 신호

#### ⚙️ 일반 거래 설정
```
과매도 임계값: 30
과매수 임계값: 70
```

#### 🎯 2배 레버리지 권장 설정
```
과매도 임계값: 35-40 (더 보수적)
과매수 임계값: 60-65 (더 민감하게)
```

#### 📝 조정 이유
- 레버리지 사용 시 극단적인 상황까지 기다리면 손실이 2배로 커짐
- 더 빠른 진입/청산으로 리스크 관리
- 안전 마진을 확보하여 청산 위험 감소

---

### 2. MACD (Moving Average Convergence Divergence)

#### 📌 지표 의미
- **구성**: MACD 선, 시그널 선, 히스토그램
- **목적**: 추세의 방향과 강도를 파악
- **계산**: 단기 EMA(12) - 장기 EMA(26), 시그널선은 MACD의 9일 EMA

#### 💡 해석 방법
- **MACD > 시그널**: 상승 추세, 매수 신호
- **MACD < 시그널**: 하락 추세, 매도 신호
- **골든 크로스**: MACD가 시그널을 상향 돌파 → 강력한 매수 신호
- **데드 크로스**: MACD가 시그널을 하향 돌파 → 강력한 매도 신호
- **히스토그램**: 두 선의 차이, 추세 강도 표시

#### ⚙️ 일반 거래 설정
```
단기 EMA: 12일
장기 EMA: 26일
시그널: 9일
```

#### 🎯 2배 레버리지 권장 설정
```
단기 EMA: 8일 (더 빠른 반응)
장기 EMA: 20일 (더 빠른 반응)
시그널: 9일 (유지)
```

#### 📝 조정 이유
- 더 빠른 트렌드 변화 감지
- 레버리지로 인한 급격한 손실 방지
- 초기 신호에 더 민감하게 반응

---

### 3. SMA 20 (Simple Moving Average 20일)

#### 📌 지표 의미
- **정의**: 최근 20일간의 종가를 단순 평균
- **목적**: 단기 추세 방향 파악
- **역할**: 지지선 또는 저항선 역할

#### 💡 해석 방법
- **현재 가격 > SMA 20**: 상승 추세, SMA가 지지선
- **현재 가격 < SMA 20**: 하락 추세, SMA가 저항선
- **가격이 SMA를 돌파**: 추세 전환 신호
- **SMA의 기울기**: 추세의 강도 표시

#### 🎯 2배 레버리지 활용 전략
```
- SMA와 현재 가격 간격이 2% 이상일 때 진입 주의
- SMA 돌파 시 확인 캔들 대기 (거짓 돌파 방지)
- SMA를 손절매 기준선으로 활용 (SMA 아래로 1-2% 설정)
```

#### 📝 레버리지 고려사항
- SMA 근처에서만 진입 (너무 멀리 진입 시 손절매 폭 증가)
- SMA 기울기가 명확할 때만 거래 (횡보 시 거래 자제)

---

### 4. Bollinger Bands (볼린저 밴드)

#### 📌 지표 의미
- **구성**: 중심선(SMA 20), 상단 밴드, 하단 밴드
- **계산**: 중심선 ± (표준편차 × 2)
- **목적**: 가격 변동성과 과매수/과매도 판단

#### 💡 해석 방법
- **상단 밴드 접근**: 과매수 가능성, 저항선
- **하단 밴드 접근**: 과매도 가능성, 지지선
- **밴드 폭 좁아짐**: 변동성 감소, 큰 움직임 전조
- **밴드 폭 넓어짐**: 변동성 증가, 추세 강화
- **밴드 돌파**: 강한 추세 또는 반전 신호

#### ⚙️ 일반 거래 설정
```
기간: 20일
표준편차 배수: 2.0
```

#### 🎯 2배 레버리지 권장 설정
```
기간: 20일 (유지)
표준편차 배수: 1.5 (더 타이트하게)
```

#### 📝 조정 이유
- 더 빠른 매매 신호 획득
- 밴드 접촉 빈도 증가로 더 많은 진입/청산 기회
- 변동성 확대 전에 미리 대응

---

## ATR 기반 손절매 조정

### ATR (Average True Range) - 평균 진폭

#### 📌 지표 의미
- **정의**: 일정 기간 동안의 평균 가격 변동폭
- **목적**: 변동성 측정 및 손절매 설정
- **계산**: 최고가-최저가, 최고가-전일종가, 전일종가-최저가 중 최대값의 평균

#### 💡 활용 방법
- **변동성 측정**: ATR이 클수록 변동성 높음
- **손절매 설정**: ATR의 배수로 손절매 위치 결정
- **포지션 사이징**: ATR에 따라 포지션 크기 조정

#### ⚙️ 일반 거래 설정
```python
롱 포지션 손절매 = 진입가 - (ATR × 2.0)
숏 포지션 손절매 = 진입가 + (ATR × 2.0)
```

#### 🎯 2배 레버리지 권장 설정
```python
롱 포지션 손절매 = 진입가 - (ATR × 1.0-1.5)
숏 포지션 손절매 = 진입가 + (ATR × 1.0-1.5)
```

#### 📝 조정 이유
- **레버리지 손실 증폭**: 2% 손실이 실제 4% 손실이 됨
- **청산 리스크 감소**: 더 타이트한 손절로 청산 위험 최소화
- **자본 보존**: 한 번의 손실로 큰 타격을 입지 않도록 방어

---

## 포지션 사이즈 및 리스크 관리

### 1. 포지션 사이즈 조정

#### ⚙️ 일반 거래 설정
```
단일 포지션: 포트폴리오의 1-5%
최대 리스크: 거래당 2%
```

#### 🎯 2배 레버리지 권장 설정
```
단일 포지션: 포트폴리오의 0.5-2.5% (50% 감소)
최대 리스크: 거래당 1% (50% 감소)
이유: 레버리지로 실제 익스포저가 2배이므로
```

### 2. 리스크 임계값 조정

#### ⚙️ 일반 거래 설정
```
신뢰도 임계값: 0.7 (70% 확신)
리스크 임계값: 0.8 (80% 이하만 허용)
```

#### 🎯 2배 레버리지 권장 설정
```
신뢰도 임계값: 0.75-0.8 (75-80% 확신)
리스크 임계값: 0.6-0.7 (60-70% 이하만 허용)
```

### 3. ADX 기반 추세 필터

#### 📌 ADX (Average Directional Index)
- **범위**: 0 ~ 100
- **목적**: 추세의 강도 측정 (방향 무관)

#### ⚙️ 일반 거래 설정
```
ADX < 25: 추세 없음 또는 약한 추세
ADX 25-50: 보통 추세
ADX > 50: 강한 추세
```

#### 🎯 2배 레버리지 권장 설정
```
ADX < 30: 거래 자제 (강한 추세에서만 거래)
ADX 30-50: 포지션 사이즈 축소
ADX > 50: 정상 포지션 사이즈 (최대 2.5%)
```

#### 📝 이유
- 레버리지는 강한 추세에서만 유리
- 횡보 장세에서는 손절 연속 발생 위험

## 통합 전략 예시

### 매수 진입 조건 (2배 레버리지)

✅ **모든 조건 충족 시만 진입**

1. **추세 확인**
   - ADX > 30 (강한 추세)
   - 가격 > SMA 20 (상승 추세)
   - SMA 20 기울기 상승

2. **모멘텀 확인**
   - RSI: 35-60 범위 (과매도/과매수 제외)
   - MACD > 시그널 (상승 모멘텀)
   - MACD 히스토그램 증가 중

3. **변동성 확인**
   - 볼린저 밴드 폭 확대 중 (추세 강화)
   - 가격이 하단 밴드 근처에서 반등 (매수 타이밍)
   - 거래량 증가 (매수 압력 확인)

4. **리스크 관리**
   - 포지션 사이즈: 포트폴리오의 1-2%
   - 손절매: 진입가 - (ATR × 1.2)
   - 목표가: 진입가 + (ATR × 2.5-3.0)
   - Risk/Reward Ratio: 최소 1:2

### 매도 청산 조건

⚠️ **다음 중 하나라도 해당 시 즉시 청산**

1. **손절매 도달**: 진입가 - (ATR × 1.2)
2. **RSI > 65**: 과매수 진입
3. **MACD < 시그널**: 데드 크로스
4. **가격 < SMA 20**: 추세 이탈
5. **목표가 도달**: 진입가 + (ATR × 2.5)
6. **ADX 하락**: 추세 약화

## 레버리지 거래 핵심 원칙 📋

### ✅ DO (해야 할 것)

1. **더 타이트한 손절매**
   - ATR 배수: 2.0 → 1.0-1.5로 감소
   - 손실 제한: 거래당 1% 이하

2. **더 보수적인 진입**
   - RSI 임계값: 30/70 → 35/65
   - ADX 최소값: 25 → 30으로 상향
   - 신뢰도: 75% 이상만 거래

3. **더 작은 포지션**
   - 포지션 사이즈: 기존의 50%
   - 최대 동시 포지션: 3개 이하

4. **더 강한 추세에서만**
   - ADX > 30인 시장에서만 거래
   - 명확한 추세 방향 확인

5. **더 빠른 반응**
   - MACD 파라미터 단축 (8, 20, 9)
   - 볼린저 밴드 표준편차 축소 (1.5)

### ❌ DON'T (하지 말아야 할 것)

1. **과도한 레버리지 중첩 금지**
   - 여러 포지션에 동시 레버리지 사용 자제
   - 상관관계 높은 자산 동시 거래 금지

2. **약한 추세에서 거래 금지**
   - ADX < 30인 횡보장 거래 금지
   - 불명확한 신호에 진입 금지

3. **손절매 이동 금지**
   - 한번 설정한 손절매는 절대 확대 금지
   - 희망적 사고로 손실 방치 금지

4. **감정적 거래 금지**
   - 복수 매매 금지
   - 연속 손실 후 포지션 확대 금지

5. **뉴스 기반 충동 거래 금지**
   - 중요 뉴스 발표 전후 레버리지 거래 자제
   - 변동성 극심 시 관망

## 실전 적용 단계

### 1단계: 백테스팅
- 과거 데이터로 조정된 지표 테스트
- 승률, 수익률, 최대 낙폭 확인
- 최소 3개월 이상 데이터 검증

### 2단계: 페이퍼 트레이딩
- 실제 시장에서 가상 거래 실행
- 최소 1개월 이상 연습
- 일관된 수익 확인

### 3단계: 소액 실전
- 최소 금액으로 시작 (총 자본의 5% 이하)
- 전략 검증 및 심리 테스트
- 안정적 수익 후 단계적 확대

### 4단계: 점진적 확대
- 월별 성과 검토
- 안정적 수익 시에만 확대
- 최대 자본의 20%까지만 레버리지 거래

## ⚠️ 중요 경고사항

### 레버리지 리스크

1. **손실 증폭**
   - 2% 하락 = 4% 손실 (2배 레버리지)
   - 5% 하락 = 10% 손실
   - 10% 하락 = 20% 손실

2. **청산 위험**
   - 거래소마다 청산 가격 설정
   - 급격한 변동 시 전액 손실 가능
   - 청산 가격 항상 모니터링 필수

3. **펀딩 비용**
   - 무기한 선물 거래 시 펀딩 비용 발생
   - 장기 보유 시 수익 잠식
   - 펀딩 비율 확인 필수

### 시장 상황별 대응

#### 고변동성 장세
```
- 포지션 사이즈 50% 추가 감소
- 손절매 20% 타이트하게 조정
- 레버리지 1배로 축소 또는 거래 중단
```

#### 횡보장
```
- 레버리지 거래 전면 중단
- 현물 거래 또는 관망
- ADX < 30 지속 시 휴식
```

#### 급락장
```
- 모든 레버리지 포지션 청산
- 시장 안정화까지 대기
- 무리한 역추세 거래 금지
```
